# v10 高中词库测试结果

- 日期：2026-07-03
- 范围：为“单词TUO”增加高中人教版词库，并支持与雅思词库自由切换。

## 验证结果

- 首页：切换到“高中人教版”后，今日学习摘要显示当前词库，并按高中词库计算新词和待复习数量。
- 学习队列：新词、复习、薄弱词和统计均按当前学习词库过滤。
- 词库页：新增“全部词库 / 雅思词库 / 高中人教版”筛选；可搜索、查看和编辑高中词条。
- 设置页：新增“学习词库”选择，可在“雅思词库 / 高中人教版 / 全部词库”之间切换。
- 导入：新增“导入高中词库”按钮；首次进入时如果本地没有高中词库，会自动导入内置数据。
- 持久化：刷新页面后高中词库和当前学习词库设置不丢失。
- PWA：Service Worker 已缓存高中词库 JSON 和脚本文件，支持离线继续使用已导入数据。
