# v16 每日新词修复测试结果

日期：2026-07-13

## 问题

用户反馈“每日新词一直都是 0 个”。排查后发现旧逻辑用当天所有 `new-complete` 记录扣减每日新词额度，没有按当前词库隔离；当今天在其它词库学过新词后，切回当前词库也会被扣光额度。

同时，如果当前词库本身没有未学习单词，页面只显示 0，没有告诉用户应该切换词库、导入词库或添加单词。

## 修复

- 每日新词额度改为按当前学习词库扣减。
- `全部词库` 的词库匹配逻辑修正为匹配所有单词。
- 首页新增“刷新今日新词”按钮，会清除未完成的新词会话并重新计算今日新词。
- 当前词库没有未学习单词时，提示切换词库、导入词库或添加新单词。
- PWA 资源版本升级到 v16。
