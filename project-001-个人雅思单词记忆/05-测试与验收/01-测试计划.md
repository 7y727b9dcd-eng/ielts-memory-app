# 第一版测试计划

## 自动检查

- JavaScript 与 Service Worker 语法。
- PWA 清单 JSON 结构、图标文件和离线缓存注册。
- iOS 工程名称、显示名、Bundle ID、资源目录和应用图标配置。

## 浏览器流程

使用 390 x 844 的移动端视口验证：

1. 清空本地数据后能加载 8 个示例雅思词。
2. 文本导入能新增单词并过滤重复词。
3. 搜索、编辑、刷新后持久化、删除均正确。
4. 复习页能显示答案并记录“认识”反馈。
5. 首页今日完成数随复习更新。
6. Service Worker 成功注册，控制台无错误。

## IPA 验收

- 文件名：`单词TUO-AltStore.ipa`。
- 主屏幕名称：`单词TUO`。
- Bundle ID：`com.personal.wordtuo`。
- 包含 `index.html`、`app.js`、`styles.css`、图标和完整 Web 资源。
- 能由 iPhone AltStore 导入并启动。
