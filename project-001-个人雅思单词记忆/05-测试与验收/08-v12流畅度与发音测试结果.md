# v12 流畅度与发音测试结果

- 日期：2026-07-03
- 范围：安卓端导航卡顿、词库选择与加载、词根记忆内容、音标和网络发音。

## 根因

- 原导航切换会调用全量渲染，间接重新生成首页、词库、统计、设置。
- 高中词库加入后词量超过 2500 条，词库页一次性生成所有 DOM 节点，安卓 WebView 会明显卡顿。

## 修改

- 导航切换改为 `renderActiveView`，只渲染当前页面。
- 词库页改为每次显示 40 条，点击“加载更多”继续追加。
- 词库页默认跟随当前学习词库；选择雅思只显示雅思，选择高中只显示高中，避免互相干扰。
- 学习、复习、统计、三周抽查按当前学习词库过滤。
- 单词新增 `audioUrl` 和 `lexiconFetchedAt` 字段。
- 学习或朗读当前词时在线补充音标和音频，并保存到本地。
- 发音优先播放网络音频，失败时回退到系统朗读。
- 词根词缀、联想钩子、词族记忆改为更具体的自动生成内容。
