# AltStore 发布说明

## 构建

根目录工作流 `.github/workflows/build-word-tuo-ipa.yml` 会：

1. 将当前 PWA 资源复制进 iOS 容器。
2. 用 XcodeGen 生成工程。
3. 在 macOS runner 上构建未签名应用。
4. 打包 `WordTUO-AltStore.ipa`。
5. 将构建结果发布到 `word-tuo-artifacts` 分支。

## 安装边界

生成的 IPA 不含 Apple 账号或证书。用户需在 iPhone AltStore 中选择 IPA，由 AltStore 使用用户自己的 Apple ID 完成签名。免费 Apple ID 安装通常需要定期刷新。

旧的 `ListeningTraining-AltStore.ipa` 与本项目无关，不得用于“单词TUO”验收。
