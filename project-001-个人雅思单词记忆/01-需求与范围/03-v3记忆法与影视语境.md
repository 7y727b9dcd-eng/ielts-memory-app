# v3 记忆法与影视语境

## 五种记忆法

1. 词根词缀：显示构词拆解和中文含义。
2. 联想钩子：用短句把词形、发音和词义连接起来。
3. 词族联结：展示同词根、派生和反义词。
4. 音节拼写：按拼写节奏拆分单词。
5. 语境记忆：使用例句并继续参与填空训练。

示例雅思词使用人工整理材料；其他词使用保守的本地词缀分析，必须标注自动拆解并允许编辑修正。

## 影视语境

- 每个单词最多保存3个片段。
- 支持 YouTube 链接以及 HTTPS 的 MP4/WebM 链接。
- 每段必须为5–20秒，保存标题和字幕提示。
- 只预连接视频域名和保存元数据，点击播放时才加载播放器。
- 不下载视频、不抓取影视网站、不绕过来源限制。
- 无免费搜索接口时，使用外部搜索后粘贴公开视频链接的流程。
