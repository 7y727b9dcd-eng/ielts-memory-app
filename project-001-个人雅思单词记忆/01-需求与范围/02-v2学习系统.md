# v2 学习系统需求

## 学习入口

- 首页首屏必须显示“开始记忆单词”主按钮。
- 同时显示今日新词、到期复习、今日完成和连续学习。
- 学习页包含新词、复习和抽查三个模式。

## 系统记忆

- 默认每日10个新词，可在设置中调整为1-50。
- 新词依次完成认识、主动回忆和例句填空。
- 复习使用“不认识 / 模糊 / 认识”三档反馈并由 SM-2 安排间隔。
- 学习会话和抽查均支持刷新或退出后继续。

## 三周抽查

- 每21天自动到期；到期前允许模拟抽查。
- 最多30词，混合释义选择、拼写和例句填空。
- 100分制，80分及格，保存分项成绩和错词。
- 错词当天重新进入学习队列，可只重测错词。

## 影视语境

- 每词最多返回3个来源，缓存搜索元数据7天。
- 不缓存视频，不抓取受保护内容，不使用付费接口。
- 无合法精准片段时必须标注为外部搜索或完整视频语境。
