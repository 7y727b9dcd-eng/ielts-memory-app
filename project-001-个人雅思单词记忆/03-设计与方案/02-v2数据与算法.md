# v2 数据与算法

## 数据升级

- 主存储键：`word-tuo-state-v2`；首次启动可读取 `word-tuo-state-v1`。
- 单词增加 `status`、`repetitions`、`easeFactor`、`intervalDays`、`lapseCount`、`learnedAt`。
- 状态分为未学习、学习中、复习中和已掌握。
- 备份版本为 2，恢复入口继续接受版本 1 数据。

## SM-2 映射

- 不认识：质量 1，重复次数归零，当天重新到期。
- 模糊：质量 3，保留学习进度并缩短间隔。
- 认识：质量 5，按 1天、3天、上一间隔乘难度因子递增。
- 难度因子范围 1.3-3.0；间隔达到30天或重复5次后标记已掌握。

## 抽查

- 抽词目标为薄弱词12、近21天新词10、已掌握词8，去重后由词库补足。
- 题目和用户进度保存在 `currentExam`，完成后写入 `exams`。
- 正确题不改变正常复习周期；错题变为学习中并当天到期。

## 影视语境

- `ClipProvider.search(word, limit)` 是统一来源接口。
- 当前免费实现返回 PlayPhrase、公开视频搜索和 YouGlish 三个外部入口。
- 搜索元数据保存在 IndexedDB `word-tuo-clips`，七天失效。
