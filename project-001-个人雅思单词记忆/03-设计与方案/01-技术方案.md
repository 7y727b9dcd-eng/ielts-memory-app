# 技术方案

## 架构

```text
PWA UI + 记忆逻辑 + localStorage
             ↓ 打包 Web 资源
SwiftUI App + WKWebView + 自定义 URL Scheme
             ↓ GitHub Actions / XcodeGen
未签名 IPA
             ↓ AltStore / Apple ID
iPhone 单词TUO
```

## Web 模块

- `index.html`：五个主要视图和两个编辑弹窗
- `styles.css`：移动优先、底部导航、卡片式复习
- `app.js`：状态、复习调度、导入导出、发音、渲染
- `manifest.webmanifest`、`sw.js`：Safari PWA 和离线缓存

## iOS 容器

- SwiftUI 仅负责承载 WKWebView
- 自定义 `wordtuo://app/` scheme 从 Bundle 加载 Web 文件
- 使用持久 `WKWebsiteDataStore`
- JSON 导出通过 JS bridge 调用 iOS 分享面板
- 最低 iOS 16，支持 iPhone/iPad

## 发布标识

- 产品名：WordTUO
