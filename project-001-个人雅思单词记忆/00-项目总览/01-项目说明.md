# 个人雅思单词记忆项目说明

- 项目编号：001
- 项目目录：`project-001-个人雅思单词记忆`
- 产品显示名：单词TUO
- 创建日期：2026-06-21
- 当前状态：开发中

## 背景与目标

开发一个仅供个人使用的雅思英语单词记忆 App，适合碎片时间复习和系统学习。第一版在没有 Mac 的条件下，通过离线 PWA + iOS WKWebView 容器构建未签名 IPA，再使用 AltStore 安装到 iPhone。

## 目标用户与场景

- 目标用户：项目所有者本人
- 学习内容：雅思核心词汇，后续可导入完整现成词库
- 碎片场景：5-15 分钟完成今日到期复习
- 系统场景：按标签学习新词并完成阶段计划

## 第一版范围

- 首页：今日待复习、今日完成、总词数、连续学习、每日目标
- 复习：英文、音标、发音、中文释义、例句、三档反馈
- 词库：搜索、筛选、添加、编辑、删除、批量导入
- 统计：掌握等级、薄弱词、学习记录
- 设置：每日目标、自动发音、导出和恢复 JSON、清空数据
- 发布：PWA、本地离线资源、AltStore IPA 构建与安装说明

## 暂不包含

- App Store 上架
- 用户注册和多人数据
- 云端 AI 生成释义或例句
- iCloud 同步、Widget、推送通知
- 完整商业词典授权内容

## 最终交付物

- 单词记忆 PWA 源码
- iOS WKWebView 容器源码与 XcodeGen 配置
- GitHub Actions IPA 构建流程
- `单词TUO-AltStore.ipa`
- 安装二维码、安装说明和验收记录

## 限制与依赖

- 当前无 Mac，iOS 编译依赖 GitHub Actions macOS runner
- IPA 使用 AltStore 和个人 Apple ID 签名
- AltStore/AltServer 需要定期刷新签名
- 学习数据默认保存在设备本地

## 验收标准

1. IPA 内显示名为“单词TUO”，Bundle ID 独立于“聆听训练”。
2. iPhone 主屏幕显示“单词TUO”，App 能正常启动。
3. 首次启动有雅思示例词，可听发音、查看例句并完成复习。
4. 复习结果会更新等级和下次复习日期，重启后数据不丢。
5. 添加、编辑、删除、搜索和批量导入可用。
6. JSON 备份可导出并恢复。
7. IPA 包含完整 Web 资源，构建日志无错误。

## 关键风险

- iOS WKWebView 本地存储与文件导入行为需真机验证
- AltServer 对中文用户目录兼容性差，必须使用纯英文运行和临时路径
- 不能通过重命名 IPA 文件改变主屏幕名称，必须修改构建配置或包内 Info.plist
- 必须验包确认 Web 资源进入 `.app`，否则安装后会空白
