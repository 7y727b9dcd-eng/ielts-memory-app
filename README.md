# 听懂再回应

一款面向中文工作对话的离线优先 PWA，用于训练长段话语的在线分块、细节保持与理解后的应答组织。

## 已实现

- 8 分钟左右的首次基线测试
- 5 分钟碎片训练和约 20 分钟系统训练
- 任务交代、进度同步、会议讨论、问题解释、意见反馈五类办公室素材
- 核心意图选择、工作顺序还原、语音复述与信息点自评
- 设备普通话朗读、本地录音与回放；拒绝麦克风权限后仍可训练
- 基于最近五题的单维度自适应难度
- 稳定语段、细节召回、遗漏类型与最近训练周报
- IndexedDB 本地数据、JSON 导出、PWA 离线缓存
- 关闭状态的 `generateSession` 与 `analyzeResponse` AI 接口

旧版雅思单词数据键不会被读取或删除。

## 本地运行

需要通过本地 HTTP 服务打开，才能测试离线缓存和麦克风权限。例如在项目目录启动任意静态文件服务器，然后用 iPhone Safari 或桌面浏览器访问。

iPhone Safari 中可通过“分享 → 添加到主屏幕”安装。

## 训练边界

本软件是个人能力训练工具，不提供医学诊断或治疗。若听后处理困难突然出现或明显加重，应优先进行听力或专业评估。
