# SwiftUI 原生版后续开发流程

## 前提

当前没有 Mac，因此这里不是第一交付路线。第一阶段先维护根目录 PWA。

## 后续条件满足后的流程

1. 在 Xcode 创建 iOS SwiftUI 项目。
2. 将 `WordMemory` 目录中的 Swift 文件拖入项目。
3. 编译并修复 API 版本、签名、权限和预览问题。
4. 接入正式雅思词库 JSON。
5. 如需长期使用，再加入 iCloud、通知、小组件和 App Store 发布配置。

## 迁移原则

- PWA 中的词库 JSON 和复习规则作为原生版的数据来源。
- 先保证本地学习体验，再增加云同步。
- 不在没有编译环境时继续扩大 SwiftUI 代码范围。
